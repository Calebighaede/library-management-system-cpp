# Library Management System (C++)

## Overview
This project is a console-based Library Management System developed in C++. It allows librarians to manage books and issued records, while supporting login-based access and file-based record storage.

## Features
- User login system
- Librarian portal
- Add, update, search, and delete books
- Display all books
- Issue and return books
- View issued books by student ID
- File handling for persistent storage

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling
- Data Structures (Stack, Queue, Heap)

## How to Run
1. Compile the project using the provided makefile
2. Run the executable
3. Use the menu options to log in and manage library records
